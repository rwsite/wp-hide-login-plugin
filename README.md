# Hide Login Plugin

Плагин для WordPress, который скрывает стандартный URL входа `wp-login.php` и перенаправляет на кастомный slug.

## Основные функции

- Изменение URL страницы входа с `wp-login.php` на кастомный (по умолчанию `signin`).
- Перенаправление неавторизованных пользователей с админ-панели на страницу 404 или кастомный URL.
- Фильтрация всех ссылок на `wp-login.php` в контенте и email.
- Поддержка переводов (домен `whl`).
- Настройки в админке: кастомный slug для входа и перенаправления.

## Установка

1. Скачай и активируй плагин.
2. Перейди в **Настройки > Общие** и настрой slug в полях "Login url" и "Redirection url".
3. Сохрани изменения - появится уведомление с новым URL входа.

## Использование

- Новый URL входа: `https://your-site.com/signin` (или твой кастомный slug).
- Неавторизованные пользователи, пытающиеся зайти в `/wp-admin/`, будут перенаправлены на 404.
- Все ссылки на `wp-login.php` автоматически заменяются.

## Разработка

- Namespace: `hide_login`
- Класс: `Plugin`
- Хуки: `plugins_loaded`, `admin_init`, `wp_loaded`
- Требования: WordPress 6.4+, PHP 8.0+
